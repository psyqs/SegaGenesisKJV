## 🔧 Using SegaGenesisKJV on a Real Sega Genesis Console

If you want to run this homebrew on a real Sega Genesis, you need more than just the ROM file.

1. Flash Cartridge: A cartridge that can load custom ROMs.
2. USB or SD Card Reader: To transfer the ROM file from your PC.
3. Software to write the ROM onto the flash cartridge.

Once you have the hardware ready:

- Copy the ROM file onto your cartridge.
- Insert the cartridge into your Sega Genesis console.
- Use the controller to navigate the interface.

## 🧩 About This Homebrew

This app is built using SGDK (Sega Genesis Development Kit), a popular tool for making homebrew games and apps on this console. It combines retro gaming with a practical Bible reader. The code is written in C and optimized for smooth scrolling and simple controls.
